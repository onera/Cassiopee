Getting started

Author : Onera / DSNA

Date : 11/11/2010

This page describes how to use *Cassiopee* package for simple pre- and post-processing tasks of Computational Fluid Dynamics workflow. Functions of the package can be accessed through python scripts or through the Graphical User Interface. Examples are given for each module.

Converter

Python script GUI View of the result
# Reading data from a file in a pyTree t
import Converter.PyTree as C
t = C.convertFile2PyTree('file1.plt')
# Read another file
t2 = C.convertFile2PyTree('file2.cgns')
# Merge in one tree
t = C.merge(t, t2)
# Save full tree
C.convertPyTree2File(t, 'out.cgns')
File/Open. Select file 'file1.plt'.
File/Add. Select file 'file2.cgns'.
File/Save. Enter 'out.cgns'.

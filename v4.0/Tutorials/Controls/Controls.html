tkCassiopee : controls

Cassiopée tutorials

This page describes the tkCassiopee keyboard and mouse controls.

If you don't have a file to manipulate, try this sample file: [tryControl.cgns].

Open the file by typing: cassiopee tryControl.cgns or click on the cassiopee icon and go to the File/Open menu.

Then try the following:


Rotate the view.
+ Zoom in/out.
+ Tilt the view.
+ Translate the view.



+ Rotate the view.
+ Translate the view.
Zoom in/out.

+ Select one zone at a time (on the graphics window or in tkTree) and place the cursor exactly on a mesh point.
+ Center the view on a mesh point.
+ + Change the point of view.
+ + Tilt the view.

+ Select one zone at a time (on the graphics window or in tkTree).
+ Deactivate (hide) or reactivate (show) one zone (on the graphics window or in tkTree).
+ + Enable multiple selections.
+ + Enable multiple deactivations.

+ + Select multiple zones (on the graphics window or in tkTree).

Note: when clicking in tkTree, you may click on a zone name, a base name or on "tree". + on a base name will select all the zones of the base. + on a base name will activate/deactivate all the zones of the base.
